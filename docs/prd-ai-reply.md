## ① 목적

관리자 모드 상담 담당자가 매일 비슷한 문의에 공공기관 문체로 답변을 작성하는 부담을 줄인다. 백지에서 손으로 쓰던 방식을 **AI 초안 기반 수정·게시**로 전환한다.

- 측정 지표: 문의 1건당 답변 작성 시간 **평균 5분 → 1분 이내**

---

## ② 사용자

| 그룹 | 설명 | 노출 여부 |
|---|---|---|
| **BIPA 상담 담당자 (관리자 모드)** | 문의 검토·답변 작성·상태 관리 | ✨ 버튼·초안 **노출** |
| 일반 문의자 (사용자 모드) | 지역기업·시민·예비창업자 | ✨ 버튼·초안 **비노출** |

비고: 초안 생성 기능은 **관리자 모드 담당자 전용**. 사용자 모드에서는 어떤 형태로도 노출되지 않는다(실습 ②의 모드 분리와 일관).

## ③ 핵심 기능 (5)

1. **✨ "AI 답변 초안 만들기" 버튼**
   - 관리자 모드 각 문의 카드의 `.admin-only` 영역에 배치
   - 보조 청록(`#0EA5E9`)으로 강조

2. **초안 자동 채움 (비어있을 때만)**
   - 생성된 초안을 "관리자 답변" textarea에 **textarea가 비어있을 때 바로 채움**
   - 이미 입력 내용이 있으면 **덮어쓰기 여부를 확인**한 후 진행 (작성 중 내용 손실 방지)

3. **생성중 안내 + 중복 클릭 방지**
   - 생성 요청 동안 버튼을 비활성화하여 **중복 요청 차단**
   - 버튼 인라인 스피너 + "초안 작성 중…" 라벨로 진행 상태 표시

4. **실패 시 쉬운 한국어 안내**
   - 생성 실패 시 카드 내에 **초등학생도 이해할 수 있는 쉬운 한국어 메시지** 표시
   - 전문용어·영문·에러코드 미사용, **실패 원인 + 재시도 방법**을 포함 (재시도 가능)

5. **초안 문체 규격**
   - 공공기관 문체, **300자 이내**
   - 구조: **인사 1줄 + 안내 1~2문단 + 다음 단계 안내 1줄**

## ④ 화면 (1) — 관리자 문의 카드의 4상태

별도 페이지 없이 관리자 문의 카드(`.admin-only` 영역) 안에서 상태 전환으로 동작한다.

| 상태 | 표시 |
|---|---|
| **기본** | ✨ "AI 답변 초안 만들기" 버튼 노출 |
| **생성중** | 버튼 인라인 스피너 + "초안 작성 중…" 라벨 + 버튼 비활성 |
| **성공** | 초안이 textarea에 채워짐 → 담당자가 수정·저장 |
| **실패** | 쉬운 한국어 안내 메시지 표시 (재시도 가능) |
